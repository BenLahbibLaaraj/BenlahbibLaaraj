Welk formaat wordt er eigenlijk gebruikt om het Date object?
YYYY-MM-DDTHH:mm:ss.sssZ
Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst?
Wanneer hetzelfde object meerdere keren voorkomt bij het opbouwen van een JSON-tekst, zal het object slechts één keer worden opgenomen in de resulterende JSON-tekst.
